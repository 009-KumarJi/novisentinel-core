# Examples

Setup guides for using NoviSentinel with common AI coding tools.

| Guide | Tool |
|-------|------|
| [claude-code.md](claude-code.md) | Claude Code (Anthropic CLI) |
| [cursor.md](cursor.md) | Cursor IDE |
| [aider.md](aider.md) | Aider (terminal agent) |
| [cline.md](cline.md) | Cline (VS Code extension) |
| [python.md](python.md) | anthropic / openai Python SDK |

Any tool that accepts a custom base URL for the Anthropic or OpenAI API works the same way.
Set `ANTHROPIC_BASE_URL=http://localhost:8000` or `OPENAI_BASE_URL=http://localhost:8000/v1` and you're done.
